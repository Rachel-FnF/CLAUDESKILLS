# 주가/시총 수집 + AI 인사이트 스킬

기업의 실시간 주식 데이터를 수집하고, AI 분석으로 투자 인사이트를 도출합니다.

## 수집 대상 데이터

- 현재 주가, 전일대비 변동률
- 시가총액
- 거래량
- PER, PBR, EPS, BPS
- 52주 고가/저가
- 배당수익률
- 외국인 소진율 (국내만)

## 수집 분기 로직

### 1. 국내 상장사 (종목코드 있음)

네이버 금융에서 스크래핑합니다. API 키 불필요.

**수집 절차:**
1. 프로젝트 DB에서 종목코드(stock_code) 확인
2. 종목코드가 있으면 → `backend/collectors/stock_collector.py`의 `StockCollector` 사용
3. 엔드포인트: `GET /api/stocks/{stock_code}`
4. 수집 항목: 현재가, 변동, PER, PBR, EPS, BPS, 시가총액, 52주 고/저, 배당수익률, 외국인소진율

**직접 수집이 안 될 경우 (서버 미가동 등):**
- WebSearch로 "{기업명} 주가 시가총액" 검색
- WebFetch로 네이버 금융 페이지 접근하여 데이터 추출

### 2. 해외 상장사 (티커 있음)

Yahoo Finance 비공식 엔드포인트를 사용합니다. API 키 불필요.

**수집 절차:**
1. 티커 심볼 확인 (예: NKE, LVMH.PA, 9983.T)
2. `backend/collectors/yahoo_collector.py`의 `YahooCollector` 사용
3. 수집 항목: 현재가, 변동률, 거래량, 52주 고/저
4. 원화 환산 포함 (USD, EUR, JPY, HKD 등)

**직접 수집이 안 될 경우:**
- WebSearch로 "{company name} stock price market cap" 검색
- 검색 결과에서 Yahoo Finance, Google Finance, Bloomberg 등의 데이터 추출

### 3. 비상장사

비상장사는 공개 주가가 없으므로 대체 지표를 수집합니다.

**수집 절차:**
1. WebSearch로 "{기업명} 기업가치 밸류에이션" 또는 "{company name} valuation funding round" 검색
2. 수집 가능한 대체 지표:
   - 최근 투자 라운드 기업가치 (스타트업의 경우)
   - 추정 매출 기반 밸류에이션
   - 업계 보도자료 속 기업가치 언급
3. 찾은 정보를 정리하여 "추정 기업가치"로 표시하고, 출처와 시점을 반드시 명시

## AI 인사이트 분석 (수집 후 반드시 수행)

수집된 주가/밸류에이션 데이터를 기반으로 Claude가 직접 다음 인사이트를 분석합니다.

### 분석 항목

**1. 밸류에이션 진단**
- PER/PBR이 업종 평균 대비 고평가/저평가 판단
- WebSearch로 "{기업명} 업종 평균 PER PBR" 또는 "{industry} average PE ratio" 검색하여 동종업계와 비교
- 현재 주가가 52주 고/저 대비 어느 위치인지 (상위 몇 %)

**2. 주가 모멘텀 판단**
- 전일대비 변동률과 최근 추세 해석
- 52주 고가 대비 현재 괴리율 → 상승 여력 또는 과열 판단
- 거래량 이상 징후 (평균 대비 급증/급감 여부)

**3. 배당/주주환원 분석** (데이터 있는 경우)
- 배당수익률 vs 시장 평균 비교
- 배당성향 추정 (EPS 대비)

**4. 외국인/수급 시그널** (국내 상장사)
- 외국인 소진율 수준 해석 (높음 → 글로벌 관심도, 낮음 → 저평가 가능성)

**5. M&A 관점 시사점**
- 시가총액 규모 대비 인수 가능성
- PBR 기준 자산가치 대비 시장 평가
- 저평가 시 인수 매력도 상승 여부

## 출력 형식

```
## 주가/시총 정보: {기업명}

### 수집 데이터
| 항목 | 값 |
|------|-----|
| 현재가 | ₩00,000 (또는 $00.00) |
| 전일대비 | +0.0% |
| 시가총액 | ₩0조 0,000억 |
| PER | 00.0 |
| PBR | 0.00 |
| 52주 고가 | ₩00,000 |
| 52주 저가 | ₩00,000 |
| 배당수익률 | 0.0% |
| 외국인 소진율 | 00.0% |

### AI 인사이트

#### 밸류에이션 진단
- (PER/PBR 업종 비교 분석, 고평가/저평가 판단과 근거)

#### 주가 모멘텀
- (52주 범위 내 위치, 추세 해석)

#### 수급/배당 시그널
- (외국인 동향, 배당 매력도 등)

#### M&A 관점
- (시총 규모, 자산가치 대비 시장 평가, 인수 매력도)

출처: 네이버 금융 / Yahoo Finance / WebSearch
수집 시각: YYYY-MM-DD HH:MM
```
